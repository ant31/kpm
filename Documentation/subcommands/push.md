# kpm push


## Options
```
usage: kpm push [-h] [--output {text,json}] [-H [REGISTRY_HOST]]
                [-o [ORGANIZATION]] [-f]

optional arguments:
  -h, --help            show this help message and exit
  --output {text,json}  output format
  -H [REGISTRY_HOST], --registry-host [REGISTRY_HOST]
                        registry API url
  -o [ORGANIZATION], --organization [ORGANIZATION]
                        push to another organization
  -f, --force           force push
```
