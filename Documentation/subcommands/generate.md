# kpm generate


## Options
```
usage: kpm generate [-h] [--output {text,json}] [--namespace [NAMESPACE]]
                    [-x VARIABLES] [-p PULL] [-H [REGISTRY_HOST]]
                    [-v [VERSION]]

optional arguments:
  -h, --help            show this help message and exit
  --output {text,json}  output format
  --namespace [NAMESPACE]
                        kubernetes namespace
  -x VARIABLES, --variables VARIABLES
                        variables
  -p PULL, --pull PULL  Fetch package from the registry
  -H [REGISTRY_HOST], --registry-host [REGISTRY_HOST]
                        registry API url
  -v [VERSION], --version [VERSION]
                        package version
```
