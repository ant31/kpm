# kpm list


## Options
```
usage: kpm list [-h] [--output {text,json}] [-u [USER]] [-o [ORGANIZATION]]
                [-H [REGISTRY_HOST]]

optional arguments:
  -h, --help            show this help message and exit
  --output {text,json}  output format
  -u [USER], --user [USER]
                        list packages owned by USER
  -o [ORGANIZATION], --organization [ORGANIZATION]
                        list ORGANIZATION packages
  -H [REGISTRY_HOST], --registry-host [REGISTRY_HOST]
                        registry API url
```
