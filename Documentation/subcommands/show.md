# kpm show


## Options
```
usage: kpm show [-h] [--output {text,json}] [--tree] [-f [FILE]]
                [-v [VERSION]] [-H [REGISTRY_HOST]]
                package

positional arguments:
  package               package-name

optional arguments:
  -h, --help            show this help message and exit
  --output {text,json}  output format
  --tree                List files inside the package
  -f [FILE], --file [FILE]
                        Display a file
  -v [VERSION], --version [VERSION]
                        package version
  -H [REGISTRY_HOST], --registry-host [REGISTRY_HOST]
                        registry API url
```
